optional step if you want to add colors:
colors are encoded as a hash # followed by 6 digits. #000000 the first two digits are red, the second pair are green, the third pair are blue. lets try changing the background color using the .css file. Open the css file in TextPad. The code in the .css wont look familiar. Cascading style sheets use another type of language than html, they use the "css" language. You don't need to learn much about css. Lets just change the

background-color: #FFFFFF

to background-color: #FF0000; what happened? the background turned red. now try #00ff00, #0000ff. What color is #000000? These colors work like light (not pigment). If all the colors are strong (FF) then you get white, if all the colors are weak, then you get black. Here is the scale (after 9 it goes to letters)

0123456789abcdef
darkest - > brightest

Try #cccccc #000044 #ccccff

Experiment with different combinations in the 6 digits to find a color you like. If you are familiar with PhotoShop or other photo editing applications you can select a color using their color wheel and they will give you the webpage code for it.
